Classtransport::MountainBike

MountainBike is an implementation of a transport::Bicycle providing a bike for cycling on rough terrain. Mountain bikes are pretty cool because they have stuff like Suspension (and you can even adjust it using SetSuspension). If you're looking for a bike for use on the road, you might be better off using a transport::RacingBike though.
